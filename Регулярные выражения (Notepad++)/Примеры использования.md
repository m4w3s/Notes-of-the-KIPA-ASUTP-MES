# Регулярные выражения для поиска и замены:  
  
- Пустая строка через 2 абзаца:  
`\r\n\r\n`  
![image (6)](https://github.com/user-attachments/assets/fa8410e2-4a57-493b-b177-cd252ed989e2)  

- Числа в формате {123} любой длины  
`\{\d+\}`  
![image (7)](https://github.com/user-attachments/assets/d5e70d6c-394a-4e27-ae40-071aef73e1c3)
  
- Строка, содержащая одну звездочку  
`\*\r\n`  
![image (8)](https://github.com/user-attachments/assets/6f438c33-8d67-4dc0-b781-74120840bba0)
  
- Строка заканчивается на число любой длины  
Найти: `\d+\r\n`  
Заменить на: `\r\n`  
![image (9)](https://github.com/user-attachments/assets/bda8e6e0-8882-4c10-aa22-fc6cfff6848a)

  
- Удалить подстроку, начинающуюся с символа нижнего подчеркивания и любого числа до конца строки  
Пример: F_AV_BRK_TOF_2;01.05.2023 0:00;173.91;0  
Выделит: `_2;01.05.2023 0:00;173.91;0`  
`_(\d+)(.+?)$`  
![image (10)](https://github.com/user-attachments/assets/8d138226-53b8-4761-b63c-c62403974888)
  
- Целое и вещественное число с точкой  
`(\d+(\.\d+)?)|(\d+)`  
![image (11)](https://github.com/user-attachments/assets/d7d8f8de-d714-4897-b411-b4e1d8ca483c)
  
- Все пробелы перед началом строки  
`\r\n\s+`  
![image (12)](https://github.com/user-attachments/assets/e036f122-ddd2-4ca8-82b4-4ca113d7b5da)

  
- Любое количество пробелов  
`^\s*$`  
![image (13)](https://github.com/user-attachments/assets/23b5f501-a69e-4a8a-9f52-f0bfe2daef3c)

  
- Строки, где нет [operation], [reconcile], [dbo]  
`^(?!.*(\[operation\]|\[reconcile\]|\[dbo\])).+$`  
![image (14)](https://github.com/user-attachments/assets/4ae312a0-8b16-4d60-a007-755a9244e94b)

  Заготовка: `^(?!.*()).+$`  
  ![image (15)](https://github.com/user-attachments/assets/09df5d3b-ea75-498a-b78a-1d30f0c2678f)
  
- Строка содержит фигурную скобку ( } ), но без запятой-разделителя. Использовал для поиска пропусков разделителей блоков JSON файлов  
`^}(?!.*,)`  
![image (16)](https://github.com/user-attachments/assets/065c8922-6bb4-4528-8d8f-5d5298b12a3a)
  
- Символ табуляции заменить на символ абзаца  
Найти: `\t`  
Заменить на: `\r\n`  
  
- Любая строка до символа =, заканчивающегося концом строки  
`^(.+?) = \r\n`  
  
- Любой текст между /* и */  
`/\*(.+?)\*/`  
  
- Подстрока Field_ID в строке  
`^(.+?)Field_ID(.+?)\r\n`  
  
- Заменить формат времени 0:00 на 00:00:00  
Найти: `(\b\d):(\d{2}\b)`  
Заменить на: `0$1:$2:00`  
  
- Заменить формат даты и времени 0000-00-00 00:00:00 на 00.00.0000 00:00 (с потерей секунд)  
Найти: `(\d{4})-(\d{2})-(\d{2}) (\d{2}):(\d{2}):(\d{2})`  
Заменить на: `$3.$2.$1 $4:$5`  
  
- Поменять местами в строке "_Month_NiQty" подстроку Month с любой последовательностью символов до конца  
Найти: `(_Month)(_\w+)`  
Заменить на: `\2\1`  
  
- Удалить кавычки возле четырехзначного числа  
Найти: `'(\d{4})'`  
Заменить на: `$1`  

- В выгрузке каталога удались весь текст, кроме расширений:  
Найти: .*(\.\w{2,7})$  
Заменить на: \1  
![1](https://github.com/user-attachments/assets/26b608a6-1c69-4b49-9194-5e9de7e7951f)

- В тексте даты к номеру дня до 10 добавить цифру 0  
Было: 1 января 2024  
Стало: 01 января 2024  
Найти: ^(\d{1})  
Заменить на: 0\1  
![image (1)](https://github.com/user-attachments/assets/aed59bce-5038-47a7-845a-0e9ff969e8bc)

- Удалить все комментарии в коде Python, начинающиеся с символа #:  
#(.+?)$  
![image (2)](https://github.com/user-attachments/assets/a8bc4f6c-28b8-4aa5-b4ce-3734ede1b181)

- Удалить пустые строки в Python с учетом табов после удаления комментариев:  
^(\s*)\r\n  
![image (3)](https://github.com/user-attachments/assets/c40ccf4d-0fd1-43d4-a97e-5e585c4f3946)

- Добавить табуляцию перед именем и расширением файла  

  Пример:  
  Исходный текст: F\Папка 1\Папка 2\Файл.pdf;31.14 Мб  
  Ожидаемый результат: F\Папка 1\Папка 2\ Файл .pdf;31.14 Мб  

  Найти: ^(.*\\)([^\\]+)(\.\w+);(.+)$  
  Заменить на: $1\t$2\t$3;\t$4  

  Примечание: Исходный текст взят из результата работы программы NikFileTree.  
  ![image (4)](https://github.com/user-attachments/assets/2c309aed-65a4-4db1-a980-976267bf1b81)

\.(?!\s)
